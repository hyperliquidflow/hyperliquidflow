V1 oklch V2 GPU V3 blur V4 sliced V5 nuclear
Variant 1
oklch interpolation
Gradient expressed in oklch() values with "in oklch" color space directive. Perceptual interpolation — best on Chrome/Firefox, may fall back silently on older WebKit.
Variant 2
GPU compositing layer
Same eased rgba stops as current production. Adds translateZ(0) + will-change: transform to force a new compositing layer. Zero visual change, sometimes fixes mobile GPU banding.
Variant 3
Blur smear
Current eased gradient + a ::after ghost copy of itself blurred 0.7px at 90% opacity. The blur dissolves hard band edges without visually softening the aura shape.
Variant 4
Stacked radials
8 separate radial-gradients, each covering only a small opacity slice (0.02–0.08). No single layer does a large alpha transition, so WebKit never has to quantise a big step.
Variant 5
Nuclear grain
Two independent noise layers: fine (1.3 freq, 5 octaves) + coarse (0.55 freq, 3 octaves). Different frequencies cancel out banding patterns the other misses. Current eased gradient underneath.
