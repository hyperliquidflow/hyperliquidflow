Goal

Multi-line chart: average net long/short bias per tier across last N snapshots. Elite going bearish while Dust goes long = the signal.

Bias formula
net_bias = (long_notional - short_notional)
          / (long_notional + short_notional)

Range: -1.0 (fully short) to +1.0 (fully long)
Deliverables
  • New API route /api/bias-trend — per-tier bias per snapshot, last 30-90 snapshots
  • Multi-line chart (recharts), one line per tier, zero reference line
  • Tier toggle checkboxes — default: Elite, Major, Large
  • KV cached 5-min TTL, invalidated by refresh-cohort
  • New page /signals/bias + nav entry under Signals

Goal

Asset x tier grid — which tiers are net long/short in which assets right now. Elite short ETH, Micro long ETH = actionable divergence at a glance.

Cell formula
bias_score = net_notional / total_oi
Color: color.green (long) / color.red (short)
Opacity: abs(bias_score), min 0.08
Deliverables

  • Cell colors from color.green / color.red at opacity = abs(bias_score)
  • Position age filter: 24h / 7d / all
  • Rightmost column: total OI per asset
  • New page /signals/heatmap + nav entry

Goal

For a selected asset, show where positions cluster on the price axis. Entry density, liquidation zones, position-heavy levels. Reveals where forced selling or profit-taking is likely.

Bucket logic
60 buckets · range: price * 0.7 to 1.3
Liq estimate: avg_px * (1 - liq_buffer) for longs
Bar width = notional (not count)
Deliverables
  • New API route /api/market-radar — bucketed long/short/liq density per price level
  • Horizontal bar chart: longs left (color.green), shorts right (color.red)
  • Current price reference line + liquidation zone overlay (dashed)
  • Asset selector (top 10 OI) + tier filter
  • Stats row: total positions, long OI, short OI, avg entries
  • New page /signals/radar + nav entry
